Petit Modbus

Project Name : Petit Modbus RTU \ Project Version : Ver 2.0 \ First Release Date : 27.07.2016 \ Last Compile Date : 27.09.2023 \ Original Author : FIRAT DEVECI \ Original Author's Web Site : www.firatdeveci.com

Description

This is very small, tiny Modbus RTU Slave library for small microcontrollers. It supports Modbus 3-6 and 16 functions! Easy to use, easy to port! Examples with PIC and ARM Microcontrollers!

Using

You only need two things:

  • Communications Send/Receive Functions
  • Tick Timer Functions

An include file called PetitModbusUserPort.h and a source file called PettiModbusPort.c are provided.

This library was tried these MCUs:

  • All of Microchip 8-16-32bit MCUs
  • STM32F0/F1/F3/F4/G3
  • MSP430
  • Atmega Series
  • Nuvoton MCUs
  • Texas DSP
